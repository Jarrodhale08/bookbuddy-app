# BookBuddy

Track your reading list, set yearly goals, log favorite quotes, join book clubs, and discover new reads based on your taste.

## Features

- **Content Library**: Browse and access learning materials
- **Progress Tracking**: Track learning progress and streaks
- **Notes & Highlights**: Save and organize learning notes
- **Reminders**: Study reminders and scheduling

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Tech Stack

- **Framework**: React Native with Expo SDK 54
- **Language**: TypeScript
- **Navigation**: Expo Router
- **Styling**: NativeWind (Tailwind CSS)
- **State Management**: Zustand
- **API Client**: Axios with React Query
